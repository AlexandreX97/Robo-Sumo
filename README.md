Robô Sumo de 500 gramas desenvolvido para a disciplina de Projeto Interdisciplinar 2.

-------------------------------------------------------

- Componentes do robô sumo:

1 Arduino Uno

1 Ponte H L298N

2 Motores DC 3~6V com caixa de redução e com 2 rodas

2 Sensores ultrassônicos HC-SR04

1 Sensor de Linha Infravermelho HW-870

4 pilhas recarregáveis AA Ni-MH

1 Suporte para 4 pilhas AA

C++
